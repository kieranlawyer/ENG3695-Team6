If you're experiencing issues with your Dbit N300 router, this section will help you diagnose and resolve common problems. From connection drops to slow speeds and login difficulties, we've outlined solutions to get your router running smoothly.


For best results, start with the basic troubleshooting steps, then explore specific issues listed below. If you continue to experience problems, consider resetting your router or checking for firmware updates.

1. Can't find the WiFi network


If you are unable to find the wifi network:

  • Ensure you router is on and connected to the internet (GREEN indicator light).
  • Attempt to establish connection via an ethernet cable. If that works, go to the router web page, and check the network name to ensure you were looking for the right one.
  • If you can connect via ethernet but cannot find the network, contact us.

2. Forgot control panel or network login details


If you are unable to log in to the network or the control panel, you can reset the router to default settings by holding the RESET button down for 10 seconds.


This will allow you to locate the network and log in using the default credentials:


SSID Name: N300-36E1

SSID Password: 123456789


Control Panel Username: admin

Control Panel Password: admin

3. Experiencing slow internet speeds


This could have many causes.


  • Restart the router to see if that resolves the issue.
  • Check with your ISP to see if they are aware of any traffic or reason for the slow connections.
  • Ensure the router is in an open area to ensure it can transmit well.
  • Consider purchasing WiFi extenders to ensure the furthest rooms in your house have access to a strong connection.

4. Devices are failing to connect to the network


This is a problem that could have mutliple solutions.


  • Ensure correct password is being entered. Consider double checking the password on the web page.
  • Try to restart the router.
  • Try to connect with an ethernet cable.
  • Check the indicator light to ensure the router is showing a connection.
  • Try out a different ethernet cable for the connection to the modem.

This should help you diagnose what exactly is failing for you. If you are still unable to connect, contact us.

5. Intermittent Internet Connection


  • Check the LAN cables and ensure everything is hooked up correctly.
  • The network could be congested. Either disconnect some of the devices using the internet or consider upgrading your bandwidth plan with your ISP. on step 6 of the user guide you can find how to limit bandwidth for certain devices.
