User Guide

This section provides step-by-step instructions to help you quickly and easily set up your Dbit N300 router. Whether you're a first-time user or configuring advanced settings, you'll find clear guidance to get your network up and running.


In this guide, you'll learn how to:

  • ✔ Unbox and connect your router
  • ✔ Access the router's web interface
  • ✔ Configure Wi-Fi settings and security
  • ✔ Set up parental controls
  • ✔ Create guest networks
  • ✔ Configure metering/bandwidth limits
  • ✔ Update firmware for optimal performance

Follow the instructions carefully, and if you encounter any issues, check the Troubleshooting or FAQ sections for additional help.
